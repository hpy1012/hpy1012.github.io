Establish blog

Veröffentlicht am 2015-10-12  |  2016-11-17

It only need takes 5 steps to establish the Blog with GitHub and Hexo,also you can have your own domain name, as you buy domain from any ICP.(This tutorial is for WINDOWS,I have no money to pay for MAC Pro -.-)

Without further delay,I’ll tell you how to do.

step 1: Install Hexo,before that you should install Git(here is how to install Git) and Node.js.(It’s so easy,download and double-click [xxx.msi]).If you have done what I said above,open CMD(WIN+R) type npm install -g hexo-cli

step 2: Build a new “Repository“,The name must be set to “username.github.io“(ex:hpy1012.github.io)

step 3: Chose a floder what you like(Ex:”F:\Blog”),right-click then chose “Git Bash Here“,type hexo init and npm install. It’s ok for your localhost configuration.You can try to debug it. Type hexo g and hexo s --debug then open your browser write Url “http://localhost:4000“. Did you see the blog? But that’s not the end.

step 4: Deploy it to the github.We should modify the “_config.yml”(root directory).Did you find deploy: and type:? Modify type:github and add repository: https://github.com/username/username.github.io and branch: master

step 5: Open Git Bash type hexo clean , hexo g , hexo d. You can access URL:”username.github.io”,it’s your blog.(Do you want have your own domain name for your blog?Buy one from a ICP,add 2 records as “@ CNAME username.github.io” and “www CNAME username.github.io”.)

If you have some advices ,contact me by Radio2Chef@gmail.com.
