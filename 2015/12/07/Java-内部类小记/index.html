Java 内部类小记

内部类用的比较少，今天在写adapter的时候用到了，记录一下。

  • 为什么要使用内部类

为什么要使用内部类？在《Think in java》中有这样一句话：使用内部类最吸引人的原因是：每个内部类都能独立地继承一个（接口的）实现，所以无论外围类是否已经继承了某个（接口的）实现，对于内部类都没有影响。

  • 怎么创建它的实例
    OuterClass.InnerClass innerClass = new OuterClass.InnerClass();
